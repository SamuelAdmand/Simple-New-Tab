# Simple New Tab - Your Bookmarks, Deserve to be Seen Again

## Project Website: [www.ainewtab.app](https://www.ainewtab.app)

Simple New Tab transforms your favorites into a new tab page, making your collected bookmarks clear, tidy, and efficient, allowing you to quickly reach the websites and resources you need most. Combined with intelligent AI search, find the answers you want faster. Supports Chrome and Edge.

## Key Features

**Set Bookmarks as New Tab Page**: After installing the extension, you can choose frequently used bookmark folders. Right-click and select "Set as Homepage" to quickly open the selected bookmark folder in the new tab page. Supports drag-and-drop sorting of bookmarks. The sidebar tree view allows you to intuitively manage bookmark folders and quickly find what you need.

**Rich Bookmark Context Menu**: Supports copying bookmark URLs, generating QR codes, and opening all bookmarks in a folder with one click.

**Sidebar Bookmarks**: Press Alt/Command + B on any web page to open the browser sidebar for quick access to bookmarks and search features.

**Floating Ball Feature**: Click the floating ball at the top right of any web page to display quick search links and default bookmarks, assisting in high-speed web browsing.

**AI Intelligent Search**: Use AI search directly on the new tab page for quick access to Doubao, Kimi, Metaso, Felo, ChatGPT, etc. It also supports classic search engines like Google and Bing, and allows searching within bookmarks and history.

**Comparison Search**: Press Cmd/Ctrl + Enter in the new tab search box to search for the same content across all search engines at once, helping you find the best answer by comparing results. Select text on any web page and use the sidebar or shortcuts (Alt + 1-7 for specific searches, Cmd/Ctrl + Enter for all) to efficiently launch a search.

**Customize New Tab Page**: Supports dark mode and customizable wallpapers. Provides 10 selected preset wallpapers and supports local uploads to personalize your new tab page.

**Browser Feature Shortcuts**: Displays shortcuts for browser history, downloads, password management, and extension management.
